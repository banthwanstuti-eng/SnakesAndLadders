# Snakes & Ladders

A classic Snakes & Ladders board game built with C++ and SFML graphics.

## How to Compile
Open MSYS2 MinGW 64-bit terminal and run:
g++ snakes.cpp -o game.exe -lsfml-graphics -lsfml-window -lsfml-system -lsfml-audio

## How to Run
./game.exe

## Controls
- SPACE = Roll the dice
- Click the Roll button = Roll the dice
- Mouse click = Navigate menus

## Features
- 2 to 4 players
- Animated token movement
- Animated dice rolling
- All 10 snakes and 8 ladders
- Win screen
